## Pré-requisitos:

- Node.js
- Yarn

## Como utilizar:

Clone o repositório
```bash
# Abra o terminal e digite esse comando ou faça download do projeto
$ git clone git@github.com:Fekleite/move.it.git
```

Entre na pasta
```bash
$ cd move.it
```

Instale as dependências
```bash
$ yarn install
```

Inicie a aplicação
```bash
$ yarn dev
```

Abra seu navegador no endereço http://localhost:3000
